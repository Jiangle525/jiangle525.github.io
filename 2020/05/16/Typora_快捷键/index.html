    1. 视图操作快捷

    功能 快捷键
    大纲视图 Ctrl+Shift+1
    文件列表视图 Ctrl+Shift+2
    放大视图 Ctrl+Shift+‘+’
    缩小视图 Ctrl+Shift+‘-’
    源代码模式 Ctrl+/
    专注模式(当前编辑行为黑，其他行为灰色) F8
    打字机模式(光标始终在屏幕中央位置) F9

    2. 标题段落快捷键

    MarkDown支持6级标题，可以使用Ctrl+数字 指定不同层次的标题和段落

    功能 快捷键
    正文 Ctrl + 0
    1级标题 Ctrl + 1
    2级标题 Ctrl + 2
    k级标题 Ctrl + k
    6级标题 Ctrl + 6
    提升标题级别 Ctrl + ‘+’
    降低标题级别 Ctrl + ‘-

    3. 字体操作快捷键

    功能 快捷键
    字体加粗 Ctrl + B
    下划线 Ctrl + U
    倾斜 Ctrl + I
    删除线 Alt + Shift + 5

    4. 插入功能快捷键

    功能 快捷键
    插入图片 Ctrl+Shift+I
    插入表格 Ctrl+T
    插入有序列表 Ctrl+Shift+[
    插入无序列表 Ctrl+Shift+]
    插入超链接 Ctrl+K
    插入代码片 Ctrl+Shift+`
    插入代码块 Ctrl+Shift+K
    插入公式块 Ctrl+Shift+M
    插入引用块 Ctrl+Shift+Q

    5. 表格扩展快捷键

    功能 快捷键
    下方插入行 Ctrl+Enter（回车）
    删除该行 Shift+Ctrl+BackSpace（退格）
    上移该行 Shift+Ctrl+↑
    下移该行 Shift+Ctrl+↓
    左移该列 Shift+Ctrl+←
    右移该列 Shift+Ctrl+→

    6. 跳转操作

    功能 快捷键
    跳转到文首 Ctrl+Home
    Ctrl+Fn+←
    跳转到所选内容 Ctrl+J
    跳转到文末 Ctrl+End
    Ctrl+Fn+→
